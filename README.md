# Benchmark of rust-postgres

This repository contains benchmarks for `postgres` and `tokio-postgres`

## Installation and usage

    $ createdb mydb
    $ psql mydb < database.sql
    $ cp .env.example .env # edit this file afterwards
    $ bash benchmark.sh
